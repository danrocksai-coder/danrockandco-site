# Project Philosophy

This site was intentionally built to be:
- Fast
- Minimal
- Mobile-first
- SEO-friendly
- Easy to maintain
- Low-bloat
- Conversion-focused

Rather than relying on heavy frameworks or unnecessary animations, the site uses clean HTML and CSS with a focus on clarity, readability, and speed.
